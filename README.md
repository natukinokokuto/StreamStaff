# 配信管制室 START

白画面を避けるための「起動確認・選択メニュー最小版」です。
React/Viteなし。GitHub Pagesにそのまま置いて動きます。

## 使い方

1. ZIPを解凍
2. 中身をGitHubリポジトリにアップロード
3. GitHub Pagesで公開
4. `index.html` が開けばOK

## 入っているもの

- ホーム
- ゲーム選択メニュー
- ログボ31日チェック
- 常連メモ
- 配信ログ保存
- localStorage自動保存
