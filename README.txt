配信管制室 統合版

使い方:
1. ZIPを解凍
2. index.html をダブルクリックで起動
3. スマホで使う場合は、このフォルダごとGitHub Pagesなどにアップロード

入っているもの:
- ホーム: 幹・枝・葉の半自動トーク管理
- ゲーム: 動物 / 大喜利 / セリフ / いつ誰どこ何 / ボブジテン / レロレロ酒場
- チンチロ: サイコロ回転、毛筆スパーン、役表、音源演出
- ログボ: リスナー別31日チェック、達成品編集
- 常連: 達成品の箇条書き表示、メモ

音源:
assets フォルダ内に以下の音源を同梱
- glass.mp3
- koduzumi.mp3
- horn.mp3
- chime.mp3

注意:
ブラウザの仕様で、音は必ず「回す」などユーザー操作後に再生されます。
データはその端末のブラウザ内(localStorage)に保存されます。
